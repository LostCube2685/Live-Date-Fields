Privacy Policy

Overview

This add-on does not collect, store, transmit, or share any personal data.

Data usage

All functionality operates entirely within Google Docs and Google Apps Script. Any data used by the add-on remains inside the user’s document environment.

The add-on may store limited configuration data (such as field settings and update preferences) using Google Apps Script document properties. This data is:

  • Stored only within the user’s Google document context
  • Not accessible externally
  • Not transmitted outside Google infrastructure

No external services

This add-on does not use:

  • External APIs
  • Analytics tools
  • Tracking services
  • Third-party data processors

Permissions

Permissions are used solely to:

  • Read and modify the active Google Doc
  • Insert and update document content based on user actions
  • Manage field settings within the document

Data ownership

All content remains fully owned and controlled by the user.
