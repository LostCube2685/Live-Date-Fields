Privacy Policy

This add-on does not collect, store, transmit, or share any personal data.

Data usage

All functionality operates entirely within Google Docs and Google Apps Script. Any data used by the add-on remains inside the user’s document environment.

The add-on may store limited configuration data (such as field settings and update preferences) using Google Apps Script’s built-in document properties. This data is:

stored only within the user’s Google document context
not accessible externally
not transmitted to any server outside Google infrastructure
No external services

This add-on does not use:

external APIs
analytics tools
tracking services
third-party data processors
Permissions

The permissions requested are used solely to:

read and modify the active Google Doc content
insert and update document text based on user actions
manage user-defined field settings within the document
Data ownership

All content remains fully owned and controlled by the user.
